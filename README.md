WB Relative Date
============================

**Date:** 2010-07-07<br />
**Author:** Wes Baker<br />
**Mail:** wes@wesbaker.com<br />
**Web:** [http://wesbaker.com](http://wesbaker.com/)

What it Does
------------

Given a date (preferably a timestamp) it returns a relative date (e.g. 2 days ago).

Usage
--------

	{exp:wb_relative_date}{entry_date}{/exp:wb_relative_date}
